## What is mean total number of steps taken per day?

```r
library(dplyr)
data <- tbl_df(read.csv(unz("activity.zip", "activity.csv")))
stepdata <- data %>% 
     group_by(date) %>% 
     summarize(total_steps = sum(steps, na.rm=TRUE))
head(stepdata)
```
